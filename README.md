# gamma-plot-cli 

## Description

Outputs gamma-rays measurements to the terminal

## Prerequisites

* Node.js
* npm

## Usage

```bash
git clone https://github.com/sgdreamer7/gamma-plot-cli.git
npm ci
npm run start
```
